**chrono-kit** is an open source python library for time series analysis and forecasting.

This project was started in 2023 by members of ODTU YZT.

## Installation

### Dependencies

chronokit requires:

* python >= 3.8.17
* torch >= 1.13.1
* numpy >= 1.24.0
* pandas >= 1.4.0
* scipy >= 1.10.1
* matplotlib >= 3.2.1

### User Installation

    pip install chrono-kit

    import chronokit
